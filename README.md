# team-profile-generator

# Requirements
This application requires the use of 
1. Node.js. 
2. inquirer
# Description

As a manager
I want to generate a webpage that displays my team's basic info
so that I have quick access to emails and GitHub profiles.

# Instructions

1. The user will run the application (app.js) using Node.js.
2. The user will be prompted to input the number of employees
3. For each employee, the user will input the employee name, email, position, etc. 
4. The application will generate an HTML page and pdf preview of the web page.
